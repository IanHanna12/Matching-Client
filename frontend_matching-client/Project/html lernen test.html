Part 1

 

   html, css und JavaScript sind für die Frontendentwicklung


 In html erstellen wir die Grundstruktur für das Frontend.

In css gestalten wir die html dann.

wählen aus in welcher Schriftform es geschrieben sein soll.

 JavaScript ist die Erweiterung der beiden,
    da kann man die Seite genauer beschreiben,
    wo man mit css die ganze Seite farblich markiert hat kann man bei JavaScript
    zum Beispiel nur ein Bereich färben.
